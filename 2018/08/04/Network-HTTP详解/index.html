    • дёҖиҲ¬е°ҶиҙҰеҸ·еҜҶз ҒзӯүйҮҚиҰҒдҝЎжҒҜеӯҳж”ҫдёәsessionпјҢе…¶д»–дҝЎжҒҜеҰӮйңҖиҰҒдҝқз•ҷеҲҷж”ҫеңЁcookieдёӯ

  1. еј•е…Ҙ

            3 POST еҗ‘жҢҮе®ҡиө„жәҗжҸҗдәӨж•°жҚ®иҝӣиЎҢеӨ„зҗҶиҜ·жұӮпјҢж•°жҚ®иў«еҢ…еҗ«еңЁиҜ·жұӮдҪ“дёӯпјҢеҸҜиғҪдјҡеҜјиҮҙж–°зҡ„иө„жәҗзҡ„е»әз«Ӣе’Ң/жҲ–е·Іжңүиө„жәҗзҡ„дҝ®ж”№гҖӮ

  1. httpsпјҡд»Ҙе®үе…Ёдёәзӣ®ж Үзҡ„HTTPйҖҡйҒ“пјҢеҚіHTTPдёӢеҠ е…ҘSSLеұӮгҖӮ
    • SSLпјҢе…Ёз§°Secure Socket LayerпјҢеҚіе®үе…ЁеҘ—жҺҘеӯ—еұӮпјҢдёәNetscapeжүҖз ”еҸ‘пјҢз”Ёд»ҘдҝқйҡңеңЁInternetдёҠж•°жҚ®дј иҫ“д№Ӣе®үе…ЁпјҢеҲ©з”Ёж•°жҚ®еҠ еҜҶжҠҖжңҜеҸҜзЎ®дҝқж•°жҚ®еңЁзҪ‘з»ңдёҠд№Ӣдј иҫ“иҝҮзЁӢдёӯдёҚдјҡиў«жҲӘеҸ–пјҢеҪ“еүҚзүҲжң¬дёә3.0гҖӮе®ғе·Іиў«е№ҝжіӣең°з”ЁдәҺWebжөҸи§ҲеҷЁдёҺжңҚеҠЎеҷЁд№Ӣй—ҙзҡ„иә«д»Ҫи®ӨиҜҒе’ҢеҠ еҜҶж•°жҚ®дј иҫ“гҖӮSSLеҚҸи®®дҪҚдәҺTCP/IPеҚҸи®®дёҺеҗ„з§Қеә”з”ЁеұӮеҚҸи®®д№Ӣй—ҙпјҢдёәж•°жҚ®йҖҡи®ҜжҸҗдҫӣе®үе…Ёж”ҜжҢҒгҖӮSSLеҚҸи®®еҸҜеҲҶдёәдёӨеұӮпјҡ
      • SSLи®°еҪ•еҚҸи®®(SSL Record Protocol)пјҡе®ғе»әз«ӢеңЁеҸҜйқ зҡ„дј иҫ“еҚҸи®®(еҰӮTCP)д№ӢдёҠпјҢдёәй«ҳеұӮеҚҸи®®жҸҗдҫӣж•°жҚ®е°ҒиЈ…гҖҒеҺӢзј©гҖҒеҠ еҜҶзӯүеҹәжң¬еҠҹиғҪзҡ„ж”ҜжҢҒгҖӮ
      • SSLжҸЎжүӢеҚҸи®®(SSL Handshake Protocol)пјҡе®ғе»әз«ӢеңЁSSLи®°еҪ•еҚҸи®®д№ӢдёҠпјҢз”ЁдәҺеңЁе®һйҷ…зҡ„ж•°жҚ®дј иҫ“ејҖе§ӢеүҚпјҢйҖҡи®ҜеҸҢж–№иҝӣиЎҢиә«д»Ҫи®ӨиҜҒгҖҒеҚҸе•ҶеҠ еҜҶз®—жі•гҖҒдәӨжҚўеҠ еҜҶеҜҶй’ҘзӯүгҖӮ
    • TLSпјҢе…Ёз§°Transport Layer SecurityпјҢдј иҫ“еұӮе®үе…ЁеҚҸи®®пјҢз”ЁдәҺдёӨдёӘеә”з”ЁзЁӢеәҸд№Ӣй—ҙжҸҗдҫӣдҝқеҜҶжҖ§е’Ңж•°жҚ®е®Ңж•ҙжҖ§гҖӮTLS 1.0жҳҜIETFеҲ¶е®ҡзҡ„дёҖз§Қж–°зҡ„еҚҸи®®пјҢе®ғе»әз«ӢеңЁSSL3.0еҚҸи®®и§„иҢғд№ӢдёҠпјҢжҳҜSSL3.0зҡ„еҗҺз»ӯзүҲжң¬пјҢеҸҜд»ҘзҗҶи§ЈдёәSSL3.1пјҢе®ғжҳҜеҶҷе…ҘдәҶRFCзҡ„гҖӮиҜҘеҚҸи®®з”ұдёӨеұӮз»„жҲҗпјҡ
      • TLS и®°еҪ•еҚҸи®®(TLS Record)
      • TLS жҸЎжүӢеҚҸи®®(TLS Handshake)
